
# Scrape and solve daily puzzle from Matiks.in in >0.6 seconds

### create venv
python -m venv venv

### activate venv
macOS/Linux: source venv/bin/activate
Windows (CMD): venv\Scripts\activate.bat
Windows (PowerShell): venv\Scripts\Activate.ps1


### install requests 
pip install requests


### run solver 
python solver.py
